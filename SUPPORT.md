# Support

## Questions And Usage Help

Use GitHub Issues for reproducible bugs:

https://github.com/WebMCP-org/use-mcp-react/issues

Include:

- Package version
- React version
- Browser and runtime environment
- MCP server type and auth mode
- Minimal reproduction or failing test when possible

## Security Issues

Do not open public issues for suspected vulnerabilities. Follow `SECURITY.md`.

## Privacy Issues

For privacy-sensitive reports, follow `SECURITY.md` and include the affected version, runtime configuration, and whether browser storage was enabled.
